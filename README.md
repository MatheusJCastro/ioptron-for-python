This library allows you to easily control in a full python environment iOptron® Mounts that support the iOptron® Mount RS-232 Command Language 2014, version 2.5 from January 15th, 2019.

This code was based on the manual available at [https://www.ioptron.com/v/ASCOM/RS-232_Command_Language2014_V2.5.pdf](https://www.ioptron.com/v/ASCOM/RS-232_Command_Language2014_V2.5.pdf)

For an example in how to operate the driver check [Driver_Example](./examples/Driver_Example.ipynb).
