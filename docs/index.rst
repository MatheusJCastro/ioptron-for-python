About
-----

| This is an unofficial library to allow easy control in a full python environment of iOptron® Mounts that support the iOptron® Mount RS-232 Command Language 2014, version 2.5 from January 15th, 2019. 

| This code was based on the manual available `here <https://www.ioptron.com/v/ASCOM/RS-232_Command_Language2014_V2.5.pdf>`_.

| For an example in how to operate the driver check `Driver_Example <https://github.com/MatheusJCastro/ioptron-for-python/blob/main/examples/Driver_Example.ipynb>`_.

The full list of available commands is listed at :ref:`controler`.
